2. Ambito e natura dell'app

Conto Mio è un'app per creare contatori, tracciare eventi conteggiati e consultare la cronologia delle variazioni. L'app non richiede un account utente e non prevede registrazione.

3. Quali dati trattiamo

3.1 Dati memorizzati localmente (sul dispositivo)

  • dati dei contatori (es. nome, valore corrente, passo, impostazioni e stati),
  • cronologia degli eventi (es. incremento/decremento/reset con timestamp),
  • impostazioni dell'app (es. lingua, tema, suono, vibrazione, preferenze annunci).

I dati sono memorizzati localmente, principalmente in SQLite e nelle preferenze di sistema (SharedPreferences).

3.2 Importazione ed esportazione dati

L'app consente di esportare i dati in un file JSON e di importare dati da un file JSON selezionato dall'utente. Queste operazioni avvengono localmente sul dispositivo e non inviano i dati ai nostri server.

3.3 Notifiche

L'app può mostrare una notifica persistente per il conteggio rapido dall'area notifiche di sistema. Su Android 13+ può essere richiesta l'autorizzazione POST_NOTIFICATIONS.

3.4 Pubblicità - Google AdMob

L'app utilizza Google AdMob. In relazione alla visualizzazione degli annunci, Google può trattare, tra gli altri:

  • identificatore pubblicitario del dispositivo,
  • indirizzo IP,
  • dati tecnici del dispositivo e del sistema operativo,
  • informazioni sulle interazioni con gli annunci.

Nell'app sono disponibili preferenze annunci (personalizzati / non personalizzati).

Dettagli: https://policies.google.com/privacy

3.5 Backup di sistema Android

Se sul dispositivo è attivo il backup di sistema Android, il sistema operativo può includere i dati locali dell'app (come database e impostazioni) nel backup dell'account o durante il trasferimento tra dispositivi.

4. Finalità e basi giuridiche (GDPR)

  • Esecuzione del servizio (art. 6(1)(b) GDPR): trattamento dei dati locali necessari al funzionamento dell'app.
  • Consenso (art. 6(1)(a) GDPR): impostazioni relative alla personalizzazione degli annunci.
  • Legittimo interesse (art. 6(1)(f) GDPR): mantenimento dell'app tramite monetizzazione pubblicitaria.

5. Condivisione dei dati

Non vendiamo i dati degli utenti. I destinatari dei dati possono includere:

  • Google (per la pubblicità AdMob),
  • fornitore del sistema operativo / servizio di backup, se il backup di sistema è abilitato dall'utente.

6. Trasferimenti fuori dallo SEE

Google può trattare dati al di fuori dello Spazio Economico Europeo. I trasferimenti avvengono secondo i meccanismi previsti dalla normativa applicabile.

7. Periodo di conservazione

  • dati locali dell'app: fino a cancellazione da parte dell'utente, sovrascrittura tramite import o disinstallazione,
  • file JSON esportati: fino a cancellazione da parte dell'utente,
  • dati legati alla pubblicità: secondo le policy di Google.

8. Diritti dell'utente

Hai il diritto di:

  • accedere ai tuoi dati,
  • rettificare i tuoi dati,
  • cancellare i tuoi dati,
  • limitare il trattamento,
  • portabilità dei dati (per i dati locali),
  • opporti al trattamento,
  • revocare il consenso (quando il consenso è la base giuridica).

Hai anche il diritto di presentare reclamo all'autorità di controllo competente.

9. Sicurezza

Applichiamo il principio di minimizzazione dei dati: l'app funziona localmente e non utilizza un nostro backend per l'archiviazione dei dati utente. Consigliamo di proteggere il dispositivo (blocco schermo) e di gestire con attenzione i file esportati.

10. Minori

L'app non è destinata ai minori di 13 anni. Se ritieni che un minore abbia fornito dati involontariamente, contattaci.

11. Modifiche alla policy

Eventuali modifiche a questa policy saranno pubblicate in questo documento con data aggiornata.
