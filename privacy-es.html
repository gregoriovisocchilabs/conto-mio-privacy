Política de privacidad - Conto Mio

Última actualización: 2026-02-16

1. Responsable del tratamiento

El responsable del tratamiento es Gregorio Visocchi Labs.
Contacto: gregoriovisocchilabs@gmail.com

2. Alcance y naturaleza de la aplicación

Conto Mio es una aplicación para crear contadores, registrar eventos contados y revisar el historial de valores. La aplicación no requiere cuenta de usuario ni registro.

3. Qué datos tratamos

3.1 Datos almacenados localmente (en el dispositivo)

  • datos de los contadores (p. ej., nombre, valor actual, paso, ajustes y estados),
  • historial de operaciones (p. ej., incremento/decremento/reinicio con marcas de tiempo),
  • ajustes de la app (p. ej., idioma, tema, sonido, vibración, preferencias de anuncios).

Los datos se almacenan localmente, principalmente en SQLite y en preferencias del sistema (SharedPreferences).

3.2 Importación y exportación de datos

La aplicación permite exportar datos a un archivo JSON e importar datos desde un archivo JSON seleccionado por el usuario. Estas operaciones se realizan localmente en el dispositivo y no envían datos a nuestros servidores.

3.3 Notificaciones

La aplicación puede mostrar una notificación persistente para contar rápidamente desde el área de notificaciones del sistema. En Android 13+ puede ser necesario el permiso POST_NOTIFICATIONS.

3.4 Publicidad - Google AdMob

La aplicación utiliza Google AdMob. En relación con la publicación de anuncios, Google puede tratar, entre otros:

  • identificador publicitario del dispositivo,
  • dirección IP,
  • datos técnicos del dispositivo y del sistema operativo,
  • información sobre interacciones con anuncios.

La aplicación incluye preferencias de anuncios (personalizados / no personalizados).

Detalles: https://policies.google.com/privacy

3.5 Copias de seguridad del sistema Android

Si la copia de seguridad de Android está activada en tu dispositivo, el sistema operativo puede incluir datos locales de la aplicación (como base de datos y ajustes) en la copia de seguridad de tu cuenta o durante la transferencia entre dispositivos.

4. Finalidades y bases jurídicas (RGPD)

  • Prestación del servicio (art. 6(1)(b) RGPD): tratamiento de datos locales necesarios para el funcionamiento de la aplicación.
  • Consentimiento (art. 6(1)(a) RGPD): ajustes relacionados con la personalización de anuncios.
  • Interés legítimo (art. 6(1)(f) RGPD): mantenimiento de la app mediante monetización publicitaria.

5. Cesión de datos

No vendemos datos de usuarios. Los destinatarios pueden incluir:

  • Google (para publicidad de AdMob),
  • proveedor del sistema operativo / servicio de copia, si el usuario tiene activadas copias de seguridad del sistema.

6. Transferencias fuera del EEE

Google puede tratar datos fuera del Espacio Económico Europeo. Las transferencias se realizan conforme a los mecanismos previstos por la ley aplicable.

7. Plazo de conservación

  • datos locales de la app: hasta que el usuario los elimine, se sobrescriban por importación o se desinstale la app,
  • archivos JSON exportados: hasta que el usuario los elimine,
  • datos relacionados con anuncios: según las políticas de Google.

8. Derechos del usuario

Tienes derecho a:

  • acceso a tus datos,
  • rectificación de tus datos,
  • supresión de tus datos,
  • limitación del tratamiento,
  • portabilidad de datos (respecto a datos locales),
  • oposición al tratamiento,
  • retirar el consentimiento (cuando el consentimiento sea la base jurídica).

También tienes derecho a presentar una reclamación ante la autoridad de protección de datos competente.

9. Seguridad

Aplicamos minimización de datos: la aplicación funciona localmente y no operamos un backend propio para almacenar datos de usuarios. Recomendamos proteger el dispositivo (bloqueo de pantalla) y manejar con cuidado los archivos exportados.

10. Menores

La aplicación no está dirigida a menores de 13 años. Si crees que un menor proporcionó datos de forma involuntaria, contáctanos.

11. Cambios en la política

Cualquier cambio en esta política se publicará en este documento con la fecha actualizada.
